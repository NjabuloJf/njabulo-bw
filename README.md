# Fredi AI Developer Dashboard

A comprehensive developer portfolio dashboard showcasing multiple project categories, code samples, and developer information.

## Features

### 1. Splash Screen
- 5-second animated splash screen with modern spinner
- Progress bar animation
- Smooth transition to main dashboard

### 2. Theme System
- Light and Dark mode support
- System preference detection
- Persistent theme storage
- Smooth theme transitions

### 3. Project Categories (10+ Categories)
- AI & Machine Learning
- Downloader Tools
- Search Engines
- Dictionaries
- Movies & Entertainment
- Education Zone
- WhatsApp Bots
- Programming Tools
- E-commerce Solutions
- Health Tech

### 4. Each Category Includes
- 5 project cards with images
- Project descriptions and tags
- Technology stack information
- Hover animations and effects

### 5. Code Samples Carousel
- Auto-rotating code examples
- Syntax highlighting for multiple languages
- Manual navigation controls
- Pause on hover functionality

### 6. Developer Profile
- Professional profile card
- Skills and expertise tags
- Statistics overview
- Online status indicator

### 7. Interactive Features
- Category auto-rotation
- Smooth scrolling
- Tooltips
- Notifications system
- Keyboard navigation
- Performance optimizations

### 8. Responsive Design
- Mobile-first approach
- Adaptive layouts
- Touch-friendly interactions
- Print styles
